# Rolo's personal dotfiles

Dotfiles aka dots powered by Makefile and GNU Stow.

## Highlights

- Less cluttered `$HOME` thanks to modest XDGBDS support by apps
- *Very* minimal neovim config, without LSP plugin as I prefer VS Code for more serious development
- [zsh-unplugged](https://github.com/mattmc3/zsh_unplugged) as the "plugin manager"
- One Dark theme everywhere

## Installation

Symlink everything by running `make`, or individually: `make <package>`.
